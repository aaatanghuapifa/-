# 🎮 哆啦A梦铜锣烧大冒险
> 创意贪吃蛇网页游戏 | 三关卡多机制体验

---

## 📖 游戏简介
《哆啦A梦铜锣烧大冒险》是一款以经典贪吃蛇玩法为基础，融合多关卡主题与创新互动机制的网页小游戏。玩家将操控哆啦A梦，在三个风格迥异的场景中收集铜锣烧、躲避障碍、与抢食的老鼠NPC竞争，挑战逐步递增的难度。

游戏采用 **HTML + CSS + JavaScript** 开发，无需额外安装软件，浏览器打开即可直接运行。

---

## 🎯 核心玩法说明
- 方向键「↑↓←→」操控哆啦A梦移动，收集场景中的铜锣烧获得分数
- 收集铜锣烧解锁下一关卡，难度随关卡提升递增
- 躲避场景障碍物，防止哆啦A梦“撞墙”导致游戏结束
- 与抢食的老鼠NPC竞争，优先抢到铜锣烧获得分数
- 挑战更高分数，体验不同关卡的创新玩法机制

## 🎨 关卡设计详解

### 第1关：木板之家 · 初遇铜锣烧
- **主题风格**：暖色调木质纹理背景，还原哆啦A梦房间场景，氛围轻松治愈
- **核心机制**：
  - 基础贪吃蛇移动控制，适配新手操作习惯
  - 前2个铜锣烧为安全收集阶段，帮助玩家熟悉移动节奏
  - 收集第3个铜锣烧后，抢食的老鼠NPC正式登场
  - 老鼠会主动向铜锣烧移动，玩家需比NPC更快抢到食物
- **难度特点**：节奏平缓，规则简单，适合新手入门

### 第2关：泳池派对 · 清凉挑战
- **主题风格**：泳池水波纹背景，清新蓝调配色，适配夏日清凉氛围
- **核心机制**：
  - 保留基础移动+抢食机制，新增泳池障碍物
  - 老鼠NPC移动速度小幅提升，抢食节奏加快
  - 铜锣烧刷新位置更分散，对玩家预判与路线规划能力要求更高
- **难度特点**：节奏明显加快，需要兼顾移动、抢食与避障三重操作

### 第3关：地穴探险 · 黑暗迷宫（创新关卡）
- **主题风格**：昏暗地穴场景，搭配恐龙骨装饰障碍物，营造神秘探险氛围
- **核心创新机制**：
  - **黑暗模式**：整个场景初始处于全黑状态，视野完全受限
  - **跟随光圈**：仅以哆啦A梦为中心，有一个随角色移动的圆形光圈，照亮周围区域
  - 铜锣烧、恐龙骨障碍物、老鼠NPC均隐藏在黑暗中，仅在光圈范围内可见
  - 老鼠NPC会在地穴中随机移动，玩家无法提前预判位置，增加未知风险
- **难度特点**：高难度探索向关卡，考验玩家的空间记忆、路线规划与临场反应能力

## 🎵 音乐与音效设计
### 1. 关卡背景音乐
- 第1关：轻松活泼的哆啦A梦主题轻音乐，营造温馨治愈氛围
- 第2关：节奏轻快的泳池风电子乐，适配清凉主题与快节奏玩法
- 第3关：低沉神秘的地穴探险BGM，烘托紧张刺激的探险氛围
- **播放逻辑**：关卡切换时BGM平滑淡入淡出，避免突兀切换；游戏暂停/结束时音乐自动暂停，重新开始时从当前关卡BGM继续播放

### 2. 互动音效反馈
- 铜锣烧收集音效：清脆的“叮”声，强化收集反馈
- 老鼠抢食音效：短促的“吱吱”声，提醒玩家竞争风险
- 碰撞/游戏结束音效：低沉的提示音，强化失败反馈
- 关卡解锁音效：轻快的旋律，给玩家明确的进度反馈

## 🛠️ 运行与开发指南
### 1. 游戏运行方式
1.  解压游戏工程文件，确保 `snake.html`、`music`、`photo` 文件夹位于同一目录下
2.  双击 `snake.html` 文件，用浏览器打开即可直接运行游戏
